# milenote
A notes application created while learning React.js.

Available here: https://milenote.herokuapp.com/

## Technologies used
**Front-end**  
* React v16 with Redux  
* ES6  
* Jest with enzyme  
* Sass  
* Baseguide (CSS framework)  
* BEM (CSS methodology)  

**Back-end**  
* Express
* Firebase

**Tooling**
* Webpack
* npm  

Heroku for app deployment.
